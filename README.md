# Music Popularity Prediction & Recommendation

## Overview
An end-to-end data science project using the Spotify API to:
- Analyze trends in song popularity.
- Train a regression model to predict track popularity.
- Recommend similar songs based on audio features.

## Tech Stack
- Python- Python • Spotify API • Spotipy
- Pandas • Scikit-Learn
- Streamlit • PostgreSQL (optional)

## Setup
1. Clone this repo
2. Create `.env` file with Spotify API credentials
3. Install dependencies:
   ```bash
